两个超大数相乘(字符串相乘)
给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式。

示例 1:

输入: num1 = "2", num2 = "3"
输出: "6"
示例 2:

输入: num1 = "123", num2 = "456"
输出: "56088"
说明：

num1 和 num2 的长度小于110。
num1 和 num2 只包含数字 0-9。
num1 和 num2 均不以零开头，除非是数字 0 本身。
不能使用任何标准库的大数类型（比如 BigInteger）或直接将输入转换为整数来处理。

很明显，超大数用Integer或者Long来装是不行的，必须要用String来进行存储，这里我采用了最笨的方法，就是模拟乘法的原理，去进行计算。

class Solution {
    public String multiply(String num1, String num2) {
        // 转化成数组，并逆向排序
        Integer[] leftCharArray = new Integer[num1.length()];
        Integer[] rightCharArray = new Integer[num2.length()];
        for (int i = 0; i < leftCharArray.length; i++) {
            leftCharArray[i] = num1.charAt(num1.length() - i - 1) - '0';
        }
        for (int i = 0; i < rightCharArray.length; i++) {
            rightCharArray[i] = num2.charAt(num2.length() - i - 1) - '0';
        }

        // 逐位相乘进行计算
        int[] allCharArray = new int[num1.length() + num2.length()];
        for (int i = 0; i < leftCharArray.length; i++) {
            for (int j = 0; j < rightCharArray.length; j++) {
                allCharArray[i + j] += leftCharArray[i] * rightCharArray[j];
            }
        }

        // 将计算后的结果进行逐步移位
        for (int i = 0; i < allCharArray.length - 1; i++) {
            Integer addNum = allCharArray[i];
            allCharArray[i + 1] += addNum / 10;
            allCharArray[i] = addNum - (addNum / 10) * 10;
        }

        // 转换成String，并且去掉多余的0
        StringBuffer sb = new StringBuffer();
        boolean zero = false;
        for (int i = 0; i < allCharArray.length; i++) {
            if (allCharArray[allCharArray.length - 1 - i] != 0) {
                zero = true;
            }
            if (zero == true){
                sb.append(allCharArray[allCharArray.length - 1 - i]);
            }
        }
        if (sb.length() == 0) {
            return "0";
        }
        return sb.toString();
    }
}
