git

git命令

查看被修改过的文件列表

git status

添加当前目录文件 git add [目录]

git add .

撤销git add

git reset

提交

git commit -m 'your comment'

更新远程fork的项目

首先，将被fork项目的源添加进来。 git remote add [upstream] [giturl] git fetch [upstream] git pull [upstream] master 相当于把远程源拉回来。 在这其中，可能会有冲突，如果有冲突，则将冲突解决掉之后，git add 冲突文件并且commit->push到服务器上。 其实这个过程就是merge的过程
